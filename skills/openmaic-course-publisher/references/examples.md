# Example prompts that should trigger this skill

- "Generate a course module on CRISPR for high school students and give me a local webpage"
- "Turn this PDF into an OpenMAIC lesson and publish it locally"
- "Create a course on Bayesian statistics with quizzes and give me the localhost page"
- "Make an AI safety module from this outline and host it on my machine"
- "Use OpenMAIC to generate an explorable lesson from these requirements"

## Recommended requirement template

When the user is vague, expand the requirement into a more structured generation brief:

- topic
- target audience
- language
- desired teaching style
- requested scene types: slides, quizzes, interactive scenes, PBL
- output expectation: local classroom webpage URL

Example brief:

```text
Generate a Chinese classroom module on probability for undergraduate beginners. Include a short lecture arc, 2 quizzes, 1 intuitive visual interactive scene, and a concise recap. Keep the tone clear and pedagogical.
```
